Framework-Agnostic Architecture
This editor uses native plugins with no React dependency. Bold, Italic, Link, and Table plugins are fully framework-agnostic. Other plugins are temporarily bridged using document.execCommand.

Welcome to Editora!

This is a framework-agnostic rich text editor built with native plugins.

Try the following features:

  • Bold, italic, underline, and strikethrough formatting
  • Insert links with the native dialog
  • Create tables using the table dialog
  • Use keyboard shortcuts (Ctrl+B, Ctrl+I, Ctrl+K, etc.)
