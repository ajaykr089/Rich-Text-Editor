# ui-react runnable examples

From repository root:

```bash
cd packages/ui-react
npm run dev:examples
```

Then open:

`http://localhost:4173`

This app demonstrates:

- quick start usage (`Button`, `Input`)
- controlled context menu usage (`anchorPoint`)
- `Form` + `useForm` workflow
- runtime theming via `ThemeProvider`
- admin-ready `DataTable` with sorting, row selection, and pagination sync
- server-state UI patterns (`Skeleton`, `Alert`, `EmptyState`, success table)
