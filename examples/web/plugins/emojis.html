Insert emojis and special characters easily

Overview

The Emojis plugin provides an easy way to insert emojis and special characters into your documents. Browse through organized emoji categories or search by keyword for quick insertion of any emoji you need.

  • ‚úì Emoji picker

  • ‚úì Search functionality
  • ‚úì Recently used tracking
  • ‚úì Skin tone variations
  • ‚úì Frequently used shortcuts
  • ‚úì Emoji preview
  • ‚úì Custom emojis support

Installation

npm install @editora/plugin-emojis

Basic Usage

import Editora from '@editora/core';
import EmojisPlugin from '@editora/plugin-emojis';

const editor = new Editora('#editor');
editor.use(EmojisPlugin);

// Open emoji picker
editor.execCommand('openEmojiPicker');

Using Emojis

Insert emojis easily:

  1. Picker: Click emoji icon to open picker
  2. Browse: Navigate through categories
  3. Search: Search by keyword or emoji name
  4. Insert: Click emoji to insert at cursor

Emoji Categories

  • Smileys: Faces and emotions
  • People: People and body parts
  • Nature: Animals and plants
  • Food: Food and drink
  • Travel: Places and transportation
  • Activities: Sports and games
  • Objects: Everyday objects
  • Symbols: Symbols and signs

API Methods

  • openEmojiPicker(): Open emoji picker dialog
  • insertEmoji(emoji): Insert specific emoji
  • getRecentEmojis(): Get recently used emojis
  • addCustomEmoji(emoji, code): Add custom emoji
