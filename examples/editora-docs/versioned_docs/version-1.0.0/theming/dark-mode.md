# Dark Mode

## Checklist

- Toolbar icons remain identifiable
- Content foreground/background contrast passes AA
- Dialogs and side panels follow theme tokens
- Focus states remain visible

## Validation

Test mixed-theme multi-instance screens and ensure local theme scoping works correctly.
