# Custom Theme

## Approach

1. Start with semantic tokens
2. Override surfaces, text, border, accent
3. Validate all plugin dialogs and side panels
4. Validate table/code readability in both modes

## Notes

Prefer variable overrides over deep selector overrides for long-term maintainability.
